This repo hols code for 25-26-p03-Experimentacion-DivideYVenceras.

## Implementation

**Divide and Conquer** algorithms share a common structure based on four main operations:
1. Dividing the problem into subproblems.
2. Recursive calls to the algorithm.
3. Combining the solutions to the subproblems to form solutions to larger problems.
4. Solving the problem when the instance is small enough.
